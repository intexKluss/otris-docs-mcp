# FeatureManager

Der FeatureManager ist eine Erweiterung in DOCUMENTS mit der bestimmte Elemente der Web-Oberfläche abgeschaltet werden, falls diese nicht benötigt oder nicht genutzt werden sollen. Diese Funktion kann allgemein für alle Benutzer oder gezielt für einen einzelnen Benutzer verwendet werden.
In dieser Dokumentation werden zunächst die Konfiguration des Systems und im Anschluss die Features beschrieben, die abgeschaltet werden können.
