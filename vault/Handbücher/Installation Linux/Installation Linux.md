# Installation unter Linux

*Stand: 16.10.2024*

documentsOS wird für Ubuntu 20.04 LTS, Ubuntu 22.04 LTS und für Ubuntu 24.04 LTS unterstützt.

Documents 5 wird ab Version 5.0f für Ubuntu 20.04 LTS und ab Version 5.0h für Ubuntu 22.04 LTS unterstützt.
Documents 6 wird ab Version 6.0.1 für Ubuntu 20.04 LTS und 22.04 LTS und ab Version 6.0.2 für Ubuntu 24.04 LTS unterstützt.

- Für eine eine einfache Installation (kein Archiv, MariaDB als Datenbank) können die Links unter Standard-Installation verwendet werden.
- Für spezielle Feature (Archiv, MySQL) bitte die Links unter Feature-Installation verwenden.

**WICHTIG für Upgrades auf documentsOS:**

- Für ein Upgrade von Documents 5 bitte die Anleitung zur Migration lesen.
- Für ein Upgrade von dem Pre-Release (Documents6 Beta) aus dem Proposed Channel bitte das Kapitel Upgrades lesen.
