# AI Toolkit

The AI Toolkit is a library for connecting documentsOS to different apis for large language models.

This api documentation serves as a reference for the api. Please refer to the [[Client SDK/index|ai toolkit manual]] for more information about configuration, usage and more examples.


## Usage

The AIToolkit class serves as the main class for talking to ai services from documentsOS. Please refer to its page on the left for more information.
