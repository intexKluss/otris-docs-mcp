# Mistral

![](./assets/service-mistral-logo.png?class=medium)

[Mistral AI](https://mistral.ai/) ist ein französisches KI-Softwareunternehmen, das eigene große Sprachmodelle unter Open Source (Apache 2.0) entwickelt. Es wurde im April 2023 von drei ehemaligen Meta und Google Deepmind Forschern gegründet.

**Firmensitz:** Paris, Frankreich


### Performanz

Mistral ist ein angesehener Anbieter auf dem GenAI-Markt und bekannt für fortschrittliche Modelle wie Mistral Large. Diese Modelle bieten eine ausgewogene Leistung und konkurrieren effektiv mit anderen führenden Diensten (OpenAI, Anthropic, VertexAI, Mixtral).

In Bezug auf die Qualität erzielt das Modell Mistral Large einen soliden Wert. Während es nicht die Spitzenposition wie GPT-4 von OpenAI erreicht, bietet es dennoch eine gute Leistung, die es in die Mitte der bewerteten Modelle positioniert.

Unter dem Aspekt der Geschwindigkeit liegt Mistral Large im unteren Bereich des Spektrums. Es ist langsamer als viele seiner Konkurrenten wie GPT-3.5 Turbo und GPT-4 von OpenAI.


### Kosten

Preislich gesehen liegt Mistral Large im mittleren bis oberen Bereich der Preisskala. Es ist teurer als einige andere Modelle, bietet aber dennoch ein adäquates Preis-Leistungs-Verhältnis. Es ist günstiger als das Spitzenmodell Claude-3-Opus von Anthropic, jedoch teurer als viele andere auf dem Markt.

Quelle Modell Vergleich: [artificialanalysis.ai](https://artificialanalysis.ai/)

## API-Key

Für die Verwendung von Mistral Modellen muss zuerst ein Mistral Account erstellt werden. Danach müssen diesem Account Zahlungsdaten hinterlegt werden.
Nach erfolgreicher Anmeldung kann, wie [hier](https://docs.mistral.ai/getting-started/quickstart/#account-setup) beschrieben, ein API Key erstellt werden.
