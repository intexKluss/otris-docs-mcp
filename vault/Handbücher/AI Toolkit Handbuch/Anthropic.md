# Anthropic

![](./assets/service-anthropic-logo.png?class=medium)

Claude ist eine Familie großer Sprachmodelle, die von [Anthropic](https://www.anthropic.com/) entwickelt wurde.
Anthropic PBC ist ein US-amerikanisches Unternehmen im Bereich künstliche Intelligenz, das 2021 von ehemaligen Mitgliedern von OpenAI gegründet wurde.

**Firmensitz:** San Francisco, Kalifornien, Vereinigte Staaten


### Performanz

Anthropic ist ein führender Anbieter auf dem GenAI-Markt und bekannt für fortschrittliche Modelle wie Claude-3. Die leistungsstärksten Modelle bieten eine hohe Qualität, vergleichbar mit anderen Diensten (OpenAI, VertexAI, Mixtral, Mistral). Betrachtet man die Qualität, so erreicht das Top-Modell Claude-3 einen hervorragenden Wert, der nur knapp hinter dem Spitzenreiter GPT-4 von OpenAI liegt.

In Bezug auf die Geschwindigkeit liegt das stärkste Modell Claude-3-Opus jedoch eher im unteren Bereich. Es ist langsamer als viele Konkurrenten wie z.B. Mixtral 8x7B und Gemini 1.5. Dennoch bieten andere Modelle von Anthropic wie Claude 3 Haiku eine deutlich höhere Geschwindigkeit und gehören damit zu den schnelleren Optionen auf dem Markt.


### Kosten

Preislich gesehen befindet sich das Modell Claude-3-Opus am oberen Ende der Preisskala und zählt damit zu den teuersten verfügbaren Modellen. Andere Modelle wie Claude 3 Haiku sind jedoch deutlich günstiger und bieten eine gute Balance zwischen Preis und Leistung.

Quelle Modell Vergleich: [artificialanalysis.ai](https://artificialanalysis.ai/)
