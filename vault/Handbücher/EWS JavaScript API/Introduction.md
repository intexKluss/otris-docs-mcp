# EWS JavaScript API


## Summary

The EWS JavaScript API enables a JANUS server, e.g., a DOCUMENTS server
to connect to an Exchange Server including the Exchange Services provided
by Office 365. This enables a JANUS server as to provide Outlook-like
features.

The EWS JavaScript API is based on the Exchange Web Service API (EWS) by
Microsoft and forwards more or less the objects and functions of the
EWS API. Details on the EWS API can be found at
[MSDN](https://msdn.microsoft.com/en-us/library/office/bb204119) and
in the book "Inside Microsoft Exchange Server 2007 Web Services" by
Microsoft Press.
