# Übersicht

Die Anzeige von Workflows erfolgt über den Konfigurationsordner **Workflow**. In der Liste werden alle vorhandenen Workflows und Snapshots gruppiert mit den wichtigsten allgemeinen Einstellungen dargestellt. Über die Suchfunktion können einzelne Einträge gezielt gesucht werden.


![admincenter-6](assets/admincenter-6.png)

Abb. 6 - Workflow Übersicht

Bei Klick auf einen Listeneintrag wird dieser Workflow bzw. Snapshot im Arbeitsbereich direkt geöffnet, siehe auch [Arbeitsbereich](workspace.html).

Außerdem stehen in der Liste alle notwendigen Aktionen bereit:

- + Workflow (Schaltfläche): Bei Klick wird der Arbeitsbereich für einen neuen Workflow geöffnet.
- Freigeben (Schaltfläche): Mit dieser Aktion wird ein Workflow freigegeben, nur freigegebene Workflows können im System verwendet werden. Die Schaltfläche ist nur aktiv, wenn mindestens ein Workflow per Checkbox markiert wird. Snapshots können nicht freigegeben werden.
- Sperren (Schaltfläche): Mit dieser Aktion wird ein Workflow gesperrt. Die Schaltfläche ist nur aktiv, wenn mindestens ein Workflow per Checkbox markiert wird. Snapshots können nicht gesperrt werden.
- Exportieren (Klappliste): Mit dieser Aktion können Workflows und Snapshots als JSON-Datei exportiert werden. Die Datei wird im Download-Verzeichnis bereitgestellt. Der Klapplisteneintrag ist nur aktiv, wenn mindestens ein Workflow bzw. Snapshot per Checkbox markiert wird.
- Löschen (Klappliste): Mit dieser Aktion können Workflows und Snapshots gelöscht werden. Es können nur Workflows gelöscht werden, die vorher gesperrt wurden. Snapshots können direkt gelöscht werden. Der Klapplisteneintrag ist nur aktiv, wenn mindestens ein Workflow bzw. Snapshot per Checkbox markiert wird.
