# XML Import

FileTypes can be created or updated by importing the XML that was created by XML Export.


## FileTypes

You can use Import XML for the XML Files created by the Extension. If the FileType contains Document Templates the folder structure of the Documents should not be changed after XML Export. Then the documents will be uptdated on Import XML.
