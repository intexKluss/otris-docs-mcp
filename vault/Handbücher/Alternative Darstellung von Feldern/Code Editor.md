# Code Editor

Der Code Editor bietet ein Formular Element zur Anzeige und Bearbeitung von Programm Code.

| Name der Felddarstellung | Documents-Version | Feldtypen |
| --- | --- | --- |
| codeEditor | ab 6.0.1 | TEXT |

| Name | Beschreibung | Standardwert | Documents-Version |
| --- | --- | --- | --- |
| language | Die Programiersprache, die das Feld unterstützen soll. Ist keine angegeben, wird diese automatisch beim erstellen des Feldes bestimmt. |  | 6.0.1 |
| lineNumbers | Die Anzahl an Zeilen, die das Feld maximal lang sein darf. |  | 6.0.1 |
