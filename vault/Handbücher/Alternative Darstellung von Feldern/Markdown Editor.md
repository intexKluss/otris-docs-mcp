# Markdown Editor

Der Markdown Editor bietet ein Formular Element zur Anzeige und Bearbeitung von Markdown. In Bearbeitungsmodus wird hierzu ein Editor bereit gestellt. Im Lesemodus wird das Markdown als html gerendert.

| Name der Felddarstellung | Documents-Version | Feldtypen |
| --- | --- | --- |
| markdown | ab 6.0.1 | TEXT |

| Name | Beschreibung | Standardwert | Documents-Version |
| --- | --- | --- | --- |
| lineNumbers | Die Anzahl an Zeilen, die das Feld im Bearbeitungsmodus maximal lang sein darf. |  | 6.0.1 |
| noBorder | Entfernt den Feld-Rahmen in der Darstellung im Lesemodus. | false | 6.0.1 |
| darkViewer | Fügt dem Feld im Lesemodus einen dunklen Hintergrund hinzu. | false | 6.0.1 |
