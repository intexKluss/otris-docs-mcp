# Exchange-Anbindung in DOCUMENTS

Mit DOCUMENTS 5.0h gibt es die Möglichkeit, einen Exchange-Server anzubinden. Das hat unter anderem den Vorteil, dass die Benutzer versendete E-Mails auch in ihrem "Gesendet"-Ordner sehen können.


## Exchange On-Premise und Exchange Online

Exchange kann auf einem eigenen Server (**On-Premise**) oder in der Cloud (**Online**) betrieben werden. Beide Lösungen lassen sich an DOCUMENTS anbinden. In den folgenden Kapiteln wird für beide Varianten beschrieben, wie DOCUMENTS konfiguriert werden muss.
