# GRPC .NET 6 DOCUMENTS Hello World

Beispielprojekt für einen minimales Hello World Beispiel für die DOCUMENTS gRPC Schnittstelle

Voraussetzungen:

- Visual Studio 2022 oder Visual Studio Code
- .NET 6

die weiteren benötigten gRPC Komponenten

- Google.Protobuf
- Grpc.Net.Client
- Grpc.Tools

lädt das Visual Studio selbstständig beim Öffnen des Projektes über NuGet herunter. Die Stubs aus den .proto Dateien werden dann auch automatisch generiert.

Download: [grpc-dot-net-hello-world.zip](assets/grpc-dot-net-hello-world.zip)
