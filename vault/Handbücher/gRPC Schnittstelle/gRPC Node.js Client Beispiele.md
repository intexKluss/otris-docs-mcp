# GRPC Node.js DOCUMENTS Hello World

Beispielprojekt für einen minimales Hello World Beispiel für die DOCUMENTS gRPC Schnittstelle

Voraussetzungen:

- Visual Studio Code
- Node.js

die weiteren benötigten gRPC Komponenten

- @grpc/grpc-js
- @grpc/proto-loader

können per `npm install` automatisch aus dem NPM Repository heruntergeladen werden. Die Stubs aus den .proto Dateien werden zur Laufzeit eingelesen.

Download: [grpc-node-hello-world.zip](assets/grpc-node-hello-world.zip)
