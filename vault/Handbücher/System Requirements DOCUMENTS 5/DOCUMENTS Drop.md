## DOCUMENTS Drop

The optional **DOCUMENTS Drop** module can only be used by users with a *Named user license*.


If **DOCUMENTS Drop** is to be used as a *desktop application or as an add-in for Microsoft Word or Microsoft Outlook*, a setup must be run on the desktop clients (only available for Microsoft Windows operating systems). The setup can be downloaded e.g. from the Web Client user menu (Addons).


To use **DOCUMENTS Drop** as an *Office 365 Web Add-In*, the DOCUMENTS web application must be made available via HTTPS.
