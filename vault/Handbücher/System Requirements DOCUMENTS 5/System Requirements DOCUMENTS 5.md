# Introduction

The following overview provides information on the minimum requirements of the **otris solution platform** (*DOCUMENTS*), these requirements also apply to the use of the  solutions **otris contract**, **otris corporate**, **otris compliance** and **otris dossier**.

If used as cloud services (**otris private cloud** or **otris public cloud**), all necessary infrastructure components are provided by otris software AG. In this case, only the system requirements for client applications must be observed.

In the case of use on the company's own servers (**on-premises**), the system requirements for on-premises installations must be observed in addition to the system requirements for client applications.
