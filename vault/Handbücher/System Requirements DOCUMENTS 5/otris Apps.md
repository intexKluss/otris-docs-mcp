## otris Apps

The following information applies to the otris apps **myFavorites**, **myContract**, **myCorporate** and **myCompliance**. These apps are available in the Apple App Store and the Google Play Store.


### Mobile devices

The following operating systems are supported:

|  | Version |
| --- | --- |
| Apple | iOS since 14.3, iPadOS since 13.6 |
| Android | since 8.1 |

The device must also satisfy the following hardware requirements:

|  | Minimum requirement |
| --- | --- |
| Processor | Current multi core processor |
| RAM | 2 GB |
| Storage | 200MB free space |


### Network access (On-Premises installations)

The DOCUMENTS web application must be accessible from the mobile device. This can be achieved either using an internal WiFi network or by making the server externally accessible or using a vpn.

**Note:** A HTTPS secured connection is strongly recommended in every case!

If the app should be synced using a cellular network, then a flat rate data contract and at least a 3G HSDPA connection is required.
