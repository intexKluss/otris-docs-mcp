# Web Services

Ab Version documentsOS 6.0.1 steht im AdminCenter die neue Erweiterung **Web Services** zur Verfügung. Mit der Erweiterung können REST-Schnittstellen
für DOCUMENTS bereitgestellt werden.

Folgende Web Services können aktuell verwendet werden:

- Documents-API: Über Web Services vom Typ Documents-API können diverse, standardmäßig bereitgestellte Endpunkte verwendet werden. Weitere Informationen siehe Documents-API.
- Script-API: Über Web Services vom Typ Skript-API können eigene Skripte erstellt werden, die selbst implementierte Funktionen ermöglichen. Dabei steht der volle Umfang der Portalskript-API zur Verfügung. Weitere Informationen siehe Skript-API.
- Webhooks: Ein Webhook ist eine HTTP-basierte Rückmeldefunktion, die eine schlanke, event-gesteuerte Kommunikation zwischen zwei Server-Anwendungen ermöglicht. Weitere Informationen siehe Webhooks.
