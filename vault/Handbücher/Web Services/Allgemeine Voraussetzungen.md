# Voraussetzungen

Für die Verwendung von Web Services gelten nachfolgende Voraussetzungen und Empfehlungen:

- Der Webserver (Tomcat) muss mit der externen Anwendung kommunizieren können.
- Im Tomcat sollte SSL aktiv sein, außerdem sollten die daraus resultierenden Einstellungen (SSL aktiv und SSL-Hostname) am Mandanten korrekt konfiguriert sein, da diese Einstellungen für die Webservice-URL von dort automatisch geladen werden.
- Auf Web Services wird generell per API-Key zugegriffen. Es ist sicherzustellen, dass die allgemeine Konfiguration für die Verwaltung von API-Keys vorgenommen wurde.
