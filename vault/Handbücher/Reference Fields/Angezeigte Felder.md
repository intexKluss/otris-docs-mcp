# Angezeigte Felder

Die Auswahl der Felder, die im Register angezeigt werden sollen, kann über Trefferlisten angegeben werden.
Dazu muss im Manager an dem entsprechendem Register dem Property `hitlist` der Name einer Trefferliste zugewiesen werden.
