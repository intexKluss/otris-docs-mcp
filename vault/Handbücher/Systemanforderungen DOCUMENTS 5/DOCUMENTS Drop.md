## DOCUMENTS Drop

Das optionale Modul **DOCUMENTS Drop** kann nur von Benutzern mit einer *Named-Benutzerlizenz* verwendet werden.


Soll **DOCUMENTS Drop** als *Desktop Anwendung bzw. als Add-In für Microsoft Word bzw. Microsoft Outlook* genutzt werden, muss auf den Desktop-Clients ein Setup ausgeführt werden (nur für Microsoft Windows Betriebssysteme verfügbar). Das Setup kann z.B. im Web Client Benutzermenü (Addons) heruntergeladen werden.


Um **DOCUMENTS Drop** als *Office 365 Web-Add-In* zu nutzen, muss die DOCUMENTS Webanwendung über HTTPS zur Verfügung gestellt werden.
