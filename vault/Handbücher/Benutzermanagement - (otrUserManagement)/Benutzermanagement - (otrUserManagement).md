# Einleitung

Ab Version **documentsOS (Build 2502)** steht ein Benutzermanagement zur Konfiguration im *AdminCenter* zur Verfügung.
Über das Benutzermanagement können Redakteure, Benutzerkonten, Zugriffsprofile und Aliase verwaltet werden.
Der Zugang zum Benutzermanagement kann über verschiedene Rollen *optional* erweitert werden.
Per *Callbacks* können diverse Anpassungen am Benutzermanagement vorgenommen werden.
