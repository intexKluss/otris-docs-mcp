# AutoTexte aus globalen Informationen

| Autotext | Beschreibung |
| --- | --- |
| %currentDate% | aktuelles Datum |
| %currentDate+x% | aktuelles Datum + x Tage |
| %currentDate-x% | aktuelles Datum - x Tage |
| %currentDate.fix% | aktuelles Datum im ISO-Format yyyymmdd |
| %currentDate.fix+x% | aktuelles Datum + x Tage im ISO-Format |
| %currentDate.fix-x% | aktuelles Datum - x Tage im ISO-Format |
| %currentDate.curloc% | aktuelles Datum in der angemeldeten Locale |
| %currentTime% | aktuelle Zeit |
| %currentTime+x% | aktuelle Zeit + x Stunden |
| %currentTime-x% | aktuelle Zeit - x Stunden |
| %currentTimestamp% | aktueller Zeitstempel |
| %currentTimestamp+x% | aktueller Zeitstempel + x Tage |
| %currentTimestamp-x% | aktueller Zeitstempel - x Tage |
| %currentTimestamp.fix% | aktueller Zeitstempel im ISO-Format yyyymmddHHMMSS |
| %currentTimestamp.fix+x% | aktueller Zeitstempel + x Tage im ISO-Format |
| %currentTimestamp.fix-x% | aktueller Zeitstempel - x Tage im ISO-Format |
| %currentTimestamp.curloc% | aktueller Zeitstempel in der angemeldeten Locale |
| %currentMinute+x% | aktuelle Minute + x Minuten; Zahl zwischen 0 und 59 |
| %currentMinute-x% | aktuelle Minute - x Minuten; Zahl zwischen 0 und 59 |
| %currentHour+x% | aktuelle Stunde + x Stunden; Zahl zwischen 0 und 23 |
| %currentHour-x% | aktuelle Stunde + x Stunden; Zahl zwischen 0 und 23 |
| %currentDay% | Zahl zwischen 1 und 31 als aktueller Tag im Monat |
| %currentMonth% | Zahl zwischen 1 und 12 als aktueller Monat im Jahr |
| %currentYear% | aktuelle Jahreszahl |
| %currentWeekday% | Zahl zwischen 0 und 6 als aktueller Tag in der Woche |
| %encoding% | Encoding |
| %systemEncoding% | System Encoding |
| %buildNo% | z.B. 1736 |
| %nr.Nummernkreisname% | Neue Nummer aus dem Nummernkreis ziehen |
| %uniqueId% | gibt eine eindeutige Nummer zurück |
| %uuid% | gibt eine eindeutige UUID zurück |
| %LF% | \n |
| %CRLF% | \r\n |
| %DIR_SEP% | \ oder / |
| %REF_SEP% | \n oder \r\n (für Referenzfelder, per Ini einstellbar) |
