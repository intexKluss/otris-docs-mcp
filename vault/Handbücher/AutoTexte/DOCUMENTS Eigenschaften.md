# DOCUMENTS Eigenschaften

Alle DOCUMENTS-Eigenschaften (Dlc-Eigenschaften, können über die folgende Notation abgefragt werden:

`%documentsOptions.<Name der Eigenschaft>%`

Wobei `<Name der Eigenschaft>` für ein beliebige Eigenschaft steht, bspw. `JobUser` oder `StandardUser`. Die technischen Namen der Eigenschaften können bspw. mit dem DOCUMENTS-Manager per `Rechtsklick -> Bezeichner ändern` abgefragt werden.
