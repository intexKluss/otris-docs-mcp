# Rollen

Standardmäßig haben **Redakteure** mit der Funktion **Documents Administrator** Zugang zum *AdminCenter*. Diese Redakteure haben nach der Aktivierung einer Erweiterung immer Vollzugriff auf die Einstellungen einer Erweiterung.

Sollen andere Benutzer zusätzlich Zugang zum AdminCenter und ggf. auf die Konfigurationsordner bestimmter Erweiterungen erhalten, kann das über **Rollen** eingestellt werden.

Jede Erweiterung hat mindestens eine Rolle **admin**, die den Vollzugriff auf die Konfiguration der Erweiterung beinhaltet. Per Klick auf eine Rolle können im entsprechenden Formular Zugriffsprofile und / oder Benutzerkonten ausgewählt werden, die zusätzlich berechtigt werden sollen. **Hinweis**: Einige Erweiterungen, wie das *Benutzermanagement* haben zusätzliche Rollen (z.B. *admin*, *edit*, bzw. *read*), die Bedeutung dieser Rollen ist der Dokumentation zur jeweiligen Erweiterung zu entnehmen.
