## Web-Client

Die Standardanwendung wird als Web-Client für Desktops und Laptops bereitgestellt. Es sollten immer aktuelle Browser-Versionen verwendet werden, neue Browser-Versionen werden zeitnah in die Qualitätssicherung aufgenommen. Folgende Browser werden unterstützt:

|  |  | Unterstützte Betriebssysteme |
| --- | --- | --- |
|  | Google Chrome | Microsoft Windows, Linux (siehe Hinweis) |
|  | Mozilla Firefox | Microsoft Windows, Linux (siehe Hinweis) |
|  | Microsoft Edge, Chromium based | Microsoft Windows, Linux (siehe Hinweis) |
|  | Apple Safari | macOS (siehe Hinweis) |

**Wichtiger Hinweis:** Für den vollen Funktionsumfang mit den Optionen **lokales Bearbeiten von Dokumenten**, **CheckIn/CheckOut von Dokumenten** und **Dokumentenvergleich** (*otris contract 4.3 und höher*) werden Browsererweiterungen verwendet. Für die Bereitstellung der Browsererweiterungen muss auf den Desktop-Clients ein Setup ausgeführt werden, welches **nur für Microsoft Windows Betriebssysteme** zur Verfügung steht. Das Setup kann z.B. im Web Client Benutzermenü (Addons) heruntergeladen werden.

Die Browsererweiterung wird mit der Installation des Setups automatisch für alle verfügbaren Browser installiert. Dabei ist eine aktive Internetverbindung erforderlich. Nach dem ersten Browserneustart muss die Erweiterung unter Umständen noch zusätzlich aktiviert werden. Dies ist eine Sicherheitsmaßname der Browserhersteller und kann normalerweise über ein Popup erfolgen. Sollte sich kein Popup öffnen kann die Erweiterung wie folgt aktiviert werden:

- Google Chrome: Aufruf von chrome://extensions/
- Microsoft Edge: Aufruf von edge://extensions/
- Mozilla Firefox: Aufruf von about:addons

Der Standard Web-Client kann auch auf mobilen Endgeräten genutzt werden, je nach Geräteart (mobile Phone bzw. Tablet) werden Darstellung und Navigation optimiert und verhalten sich anders als im Webbrowser eines Desktops Clients. Browsererweiterungen stehen hier nicht zur Verfügung. Folgende Geräte werden unterstützt:

| Gerät | Betriebssystem |
| --- | --- |
| Apple iPad | iPadOS 13.6 oder höher |
| Apple iPhone | iOS 14.3 oder höher |
