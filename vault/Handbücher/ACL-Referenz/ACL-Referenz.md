# Content Related Permissions

In DOCUMENTS können auf einfache Weise Rechte für Mappentypen gesetzt werden. Dadurch kann ein Benutzer dann entweder alle oder keine Mappen des entsprechenden Mappentypes sehen. Häufig gibt es aber auch die Anforderung, dass Benutzer nur bestimmte Mappen eines Mappentyps sehen dürfen. Aus diesem Grund wurde der [Mappenklassenschutz](./class-intro.html) und die [ACLs](./acl-intro.html) eingeführt.
