## @otr-documents/otrtranslate


# Übersetzungsservice

Ab Version documentsOS 6.0.1 steht im AdminCenter die neue Erweiterung **Übersetzungsservice** zur Verfügung. Die Erweiterung bietet eine Anbindung an verschiedene Übersetzungsschnittstellen, um Texte und Dokumente innerhalb von documentsOS automatisch zu übersetzen.

Die unterstützten Sprachen und Features sind jeweils den Implementierungen der einzelnen Services zu entnehmen.
Derzeit stehen als mögliche Übersetzungsservices **DeepL**, **Google Translate** und **Libre Translate** bereit.
