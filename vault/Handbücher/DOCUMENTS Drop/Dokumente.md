# Dokumente

Im Bereich **Dokumente** ist es möglich auf bereits in DOCUMENTS vorhandene
Dokumente zuzugreifen. Über ein Suchfeld kann per Volltextsuche eine Mappe
ausgewählt werden. Die Dokumente der ausgewählten Mappe werden dann nach den
Registern gruppiert aufgelistet.
Durch die Auswahl eines Eintrages wird die von der *Host*-Applikation abhängige Funktion ausgeführt.


### Microsoft Outlook

Das ausgewählte Dokuments wird der aktuellen E-Mail als Anhang hinzugefügt.
Sollte sich Outlook nicht im Bearbeitungsmodus befinden wird ein neue Email mit dem gewählten Dokument im Anhang erstellt.


![](figures/outlook-embed-document.png)

Abb. 12 - Dokument als E-Mail-Anhang hinzufügen


### Microsoft Word

In Word wird das ausgewählte Dokumente an der aktuellen Position eingebettet. Handelt es sich bei dem ausgewählten Dokument um ein Word-Dokument, wird der
Inhalt des Dokumentes eingefügt.


![](figures/word-embed-document.png)

Abb. 13 - Einbettung eines Word-Dokumentes


### Windows DROP-Client

Im Kontext des Windows DOCUMENTS Drop-Clients wird nach der Auswahl eines Dokumentes ein
**Speichern unter...**-Dialog angezeigt. Über den Dialog kann der lokale
Speicherort für das Dokument festgelegt werden.
