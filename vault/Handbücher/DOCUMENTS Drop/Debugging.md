# Debugging

Um die Entwickler-Tools für  DOCUMENTS Drop zu aktivieren muss im
Installationsverzeichnis ein leere Datei mit dem Dateinamen `EnableDevMode.prop`
angelegt werden.
Über **F12** können die Entwickler-Tools dann in der Anwendung bzw. dem AddIn
aktiviert werden. Mit den Entwickler-Tools ist es dann zum Beispiel möglich den
DOM zu untersuchen oder auch Javascript Kommandos direkt auszuführen.
