# Textbausteine

Über den Bereich **Textbausteine** ist es möglich, Textinhalte aus DOCUMENTS
in der *Host*-Applikation zu verwenden. Nach Auswahl einer Mappe über das
Suchfeld können die Mappenfeldwerte der ausgewählten Mappe verwendet werden.


![](figures/outlook-text-snippet.png)

Abb. 11 - Textbaustein in eine Email einfügen

In Word bzw. Outlook werden die Werte an der aktuellen Position eingesetzt.
Im Windows DOCUMENTS Drop-Client wird der ausgewählte Wert in die Zwischenablage kopiert
und kann so in jeder beliebigen Anwendung eingefügt und verwendet werden.

Über ein Skript können weitere Textbausteine hinzugefügt werden. So ist es
möglich auch allgemeine (von einer Mappe unabhängige) Textbausteine zu
erstellen.
Diese Erweiterungsmöglichkeit wird im Abschnitt
[Textbausteine erweitern](configure-extend.html#textbausteine-erweitern) erläutert.
