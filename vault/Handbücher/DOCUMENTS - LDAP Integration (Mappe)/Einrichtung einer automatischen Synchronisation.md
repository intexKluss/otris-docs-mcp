# Einrichtung einer automatischen Synchronisation

Die Synchronisation kann über ein Job-Skript automatisiert werden. Tragen Sie entsprechend ein, ob dies automatisch erfolgen soll
(Checkbox *Job aktiviert*) und legen Sie eine Uhrzeit fest, zu der die Synchronisation täglich erfolgen soll. Die Einstellung *Job aktiviert* kann nur vorgenommen werden, wenn die Option *Ldap aktivieren* angeschaltet ist, da dort eine Prüfung erfolgt, ob die Verbindnung zum LDAP-Server prinzipiell hergestellt werden kann.
