## Microsoft Windows


### Server

Only the following **Operating Systems** with 64-bit architecture are supported:

|  | Microsoft Windows Server 2016 | 2019 | 2022 | 2025 |
| --- | --- |


### Database

Only the following **Database Systems** with 64-bit architecture are supported:

|  | Microsoft SQL Server (Standard / Enterprise) 2016 (EOL 07/26) | 2017 | 2019 | 2022 | 2025 |
| --- | --- |
|  | 10.6 (LTS - EOL 06/26) | 10.11 (LTS) | 11.4 (LTS) | 11.8 (LTS) * |

* MariaDB 10.11, 11.4 or 11.8 recommended - LTS versions


### Additional components

The following components supplement the installation:

|  | Java JDK / JRE (64-bit) Version 17/21 (LTS versions from Oracle or OpenJDK) |
| --- | --- |
|  | Web-Container Tomcat Version 9.x (Delivered and updated as part of the setup by otris software AG) |
|  | Optional Apache Webserver (integration can be done by customer or ordered as service) |
|  | An installed .NET framework version 4.0 or higher is required |
