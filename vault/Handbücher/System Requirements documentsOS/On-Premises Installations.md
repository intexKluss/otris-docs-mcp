# System requirements for on-premises installations

The system requirements presented below apply to software installations on company-owned servers (on-premises).

**Note:** Basic components such as operating systems and databases or additional components are generally (unless otherwise specified) supported until the end of the manufacturer's respective standard support and not for any further support period of the manufacturer (“extended support”).
