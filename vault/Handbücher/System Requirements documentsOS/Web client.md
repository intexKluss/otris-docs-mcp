## Web client

The standard application is provided as a web client for desktops and laptops. Current browser versions should always be used, new browser versions will be included in quality assurance in a timely manner. The following browsers are supported:

|  |  | Supported operating systems |
| --- | --- | --- |
|  | Google Chrome | Microsoft Windows, Linux (see note) |
|  | Mozilla Firefox | Microsoft Windows, Linux (see note) |
|  | Microsoft Edge, Chromium based | Microsoft Windows, Linux (see note) |
|  | Apple Safari | macOS (see note) |

**Important note:** Browser extensions are used for the full range of functions with the options **local editing of documents**, **CheckIn/CheckOut of documents** and **Document comparison** (*otris contract 4.3 and higher*). For the provision of the browser extensions, a setup must be executed on the desktop clients, which is **only available for Microsoft Windows operating systems**. The setup can be downloaded e.g. in the Web Client user menu (Addons).

The browser extension is automatically installed for all available browsers when the setup is installed. This requires an active Internet connection. After the first browser restart, the extension may have to be activated additionally. This is a security measure of the browser manufacturer and can usually be done via a popup. If no popup opens, the extension can be activated as follows:

- Google Chrome: call chrome://extensions/
- Microsoft Edge: call edge://extensions/
- Mozilla Firefox: call about:addons

The standard web client can also be used on mobile devices. Depending on the type of device (mobile phone or tablet), the display and navigation are optimized and behave differently than in the web browser of a desktop client. Browser extensions are not available here. The following devices are supported:

| Device | Operating system |
| --- | --- |
| Apple iPad | iPadOS 13.6 or higher |
| Apple iPhone | iOS 14.3 or higher |
