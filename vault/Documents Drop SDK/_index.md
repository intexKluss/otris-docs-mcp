# DOCUMENTS Drop SDK

The **DOCUMENTS Drop SDK** (also known as **DocumentsNativeConnector**) provides an API to interact with different host of the documents drop applications. It is accessed using a global variable called `documentsNativeConnector`, which is available in the various add-ins and clients which support DropGadgets
