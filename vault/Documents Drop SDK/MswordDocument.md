The current Word document


### Methods


**insertText(text){Promise}**
Inserts text at cursor position into the word document

| Name | Type | Description |
| --- | --- | --- |
| text | string | The text to be inserted |


##### Returns:

| Type | Description |
| --- | --- |
| Promise | Promise which resolves after the text was inserted |
