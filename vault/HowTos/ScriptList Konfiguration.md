# Einbindungsmöglichkeiten einer ScriptList


## Globales Skript

Über die globale Eigenschaft "globalScript" kann ein Skript für die anzuzeigende Liste hinterlegt werden.


Abb. 1 - globales Script konfigurieren

Beim Klick auf den Script-Button öffnet sich diese dann im Ordnerframe.


Abb. 2 - Toolbar mit Scriptlist


## Ordner

Mit der Eigenschaft "runScript" kann an einem Ordner ein Skript hinterlegt werden.


Abb. 3 - Konfiguration am Ordner

Beim Klick auf den Ordner öffnet sich dann die ScriptList.


Abb. 4 - Anzeige im Ordner


## Outbar

An einer Outbar kann die Eigenschaft "runScript" hinterlegt werden.


Abb. 5 - Konfiguration an einer Outbar

Beim Klick auf die Outbar wird dann im Ordnerframe die Liste angezeigt.


Abb. 6 - Anzeige als Outbar
